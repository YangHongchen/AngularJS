AngularJS允许在$scope上设置包括对象在内的任何类型的数据，并且在视图中还可以展示 对象的属性。
例如，我们在MyController上创建一个person对象，这个对象只有name这一个属性： 
var myApp = angular.module("myApp", []);
myApp.controller("MyController", function($scope) {
    // scope中是可以定义很复杂的数据类型的
    $scope.person = {
        name: "yangc",
        age: 26,
        gender: "male",
        home: "Hubei"
    }
});
