过滤器

用来格式化需要展示给用户的数据。 AngularJS有很多实用的内置过滤器，同时也提 供了方便的途径可以自己创建过滤器。 
var app = angular.module("app",[]);
app.controller('DemoController', ['$scope', '$filter',
    function($scope, $filter) {
        $scope.name = $filter('lowercase')("YCASE");
    }
]);

测试程序原值：{{name}}
过滤器返回值： {{name | currency : '€'}}


【内置过滤器】：
lowercase(转小写)
uppercase(转大写)
number:2（精确到小数点后两位）
currency

currency

(将数字转换成货币格式，默认是美元 [currency]直接使用, 它也允许我们自己设置货币符号：currency : '€' 设置为欧元)

date

date过滤器可以将日期格式化成需要的格式。 AngularJS中内置了几种日期格式，如果没有 指定使用任何格式，默认会采用mediumDate格式
详细用法参考官方文档;

filter

filter过滤器可以从给定数组中选择一个子集，并将其生成一个新数组返回。这个过滤器通 常用来过滤需要进行展示的元素。例如，在做客户端搜索时，可以从一个数组中立刻过滤出所需 的结果。
这个过滤器的第一个参数可以是字符串、对象或是一个用来从数组中选择元素的函数。
